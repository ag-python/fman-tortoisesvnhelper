# TortoiseSVN Helper (Windows only) 
[fman](https://fman.io) plugin to work with TortoiseSVN

# Installation
1 Install using fman Install Plugin command

# Usage
* Invoke command window with Ctrl+Shift+P/Cmd+Shift+P and type SVN
